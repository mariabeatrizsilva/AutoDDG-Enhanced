# AutoDDG
Automated Dataset Description Generation using Large Language Models

This is the code for the paper "AutoDDG: Automated Dataset Description Generation using Large Language Models" submitted to VLDB 2025. The extended version of the paper is available [here](https://arxiv.org/abs/2502.01050).



## Installation

You can install the latest development version using pip:

```
pip install git+https://github.com/VIDA-NYU/AutoDDG@main
```

## Documentation
See our [quick-start example](./examples/quick_start.ipynb) to learn how to use AutoDDG.
